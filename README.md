# Todo List - 待办任务管理

一个中文待办任务管理应用，支持与 Excel 文件同步。提供现代 UI，包含任务列表和日历视图，支持重复任务、优先级、分类和提醒功能。

## 功能特点

- **任务管理**：创建、编辑、删除任务，支持任务内容和明细
- **重复任务**：支持按天/周/月/年重复，完成后自动创建下一次 occurrence
- **优先级**：高、中、低三种优先级标识
- **分类管理**：自定义分类，支持工作、生活、学习等类别
- **日历视图**：按月展示任务，直观查看所有任务安排
- **提醒功能**：支持提前设定提醒时间和天数
- **数据持久化**：使用 Excel 文件存储所有数据

## 技术栈

- **后端**：Express.js
- **前端**：Vanilla HTML/CSS/JavaScript
- **数据存储**：Excel (xlsx library)
- **测试**：Playwright

## 安装和运行

```bash
# 安装依赖
npm install

# 启动服务器
npm start
# 或
node server.js
```

服务器运行在 http://localhost:3000

## API 接口

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/api/todos` | 获取任务列表 (支持 dateFrom, dateTo 参数) |
| POST | `/api/todos` | 创建新任务 |
| PUT | `/api/todos/:id` | 更新任务 |
| DELETE | `/api/todos/:id` | 删除任务 |
| GET | `/api/settings` | 获取设置 |
| PUT | `/api/settings` | 更新设置 |
| POST | `/api/categories` | 添加分类 |
| PUT | `/api/categories/:oldName` | 更新分类 |
| DELETE | `/api/categories/:name` | 删除分类 |

## Excel 数据结构

| 列名 | 说明 |
|------|------|
| ID | 任务唯一标识 |
| 任务内容 | 任务标题 |
| 任务明细 | 任务详细描述 |
| 完成状态 | 未完成/已完成 |
| 创建时间 | 任务创建时间 |
| 优先级 | 高/中/低 |
| 分类 | 任务分类 |
| 截止日期 | 任务截止日期 |
| 截止时间 | 任务截止时间 |
| 完成时间 | 任务完成时间 |
| 重复类型 | 无/每天/每周/每月/每年 |
| 重复值 | 重复间隔数值 |
| 提醒时间 | 提醒时间点 |
| 提醒天数 | 提前多少天提醒 |
| 提醒小时 | 提前多少小时提醒 |

## 默认分类

- 工作
- 生活
- 学习
- 其他

默认分类不可删除。
